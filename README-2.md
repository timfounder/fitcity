# FITCITY CHIMGAN

Сайт фитнес-клуба FitCity Chimgan в Ташкенте.

## Структура

```
.
├── index.html      Разметка страницы
├── styles.css      Все стили (cinematic dark theme)
├── i18n.js         Словари переводов RU / UZ / EN
└── app.js          Анимации, калькулятор, форма, переключатель языков
```

Сайт статический — никаких сборщиков, фреймворков и зависимостей. Можно положить на любой статичный хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages, обычный nginx — что угодно.

## Возможности

- Многоязычность RU / UZ / EN с автоопределением и сохранением выбора
- Кинематичные анимации (sticky-scroll "Твой путь", split-text glitch, magnetic курсор, частицы в hero, parallax, scroll-progress)
- Калькулятор стоимости абонемента с real-time пересчётом
- Форма заявки с отправкой в Telegram-бот
- Floating Telegram-кнопка и sticky CTA для мобильных
- SEO: Open Graph, Schema.org JSON-LD (GymOrFitnessClub), полные meta-теги
- Адаптив: десктоп / планшет / мобильный
- Тёмная тема с фирменным красным акцентом

## Настройка перед запуском

### 1. Telegram-бот для заявок

Чтобы заявки с формы приходили в твой Telegram:

1. Создай бота через [@BotFather](https://t.me/BotFather) — получишь `BOT_TOKEN`
2. Создай группу/канал в Telegram, добавь бота как админа
3. Узнай `CHAT_ID` группы — например, через [@userinfobot](https://t.me/userinfobot) или [@getidsbot](https://t.me/getidsbot)
4. В файле `app.js` найди раздел `initForm()` и вставь свои значения:
   ```js
   const TELEGRAM_BOT_TOKEN = 'ВАШ_ТОКЕН';
   const TELEGRAM_CHAT_ID   = 'ВАШ_CHAT_ID';
   ```

Пока эти поля пустые, форма работает в демо-режиме (симулирует успех, заявки логируются в консоль браузера).

> ⚠️ **Безопасность.** Токен бота находится в клиентском JS, его можно извлечь. Это допустимо для prod-формы — бот ограничен правами писать только в твою группу. Для повышенной безопасности заявки можно проксировать через свой backend (Cloudflare Worker, Vercel Function).

### 2. Контактные данные

Если меняется адрес, телефон или ссылки на соцсети — правь в двух местах:
- `index.html` — в секциях `<!-- LOCATION -->` и `<!-- FOOTER -->`, а также в JSON-LD в `<head>`
- `i18n.js` — ключи `location.meta.*` для всех трёх языков

### 3. Шрифты и иконки

Шрифты подгружаются с Google Fonts (Anton, Barlow Condensed, Manrope, JetBrains Mono). Если нужен полный офлайн — скачай их и положи рядом, заменив `<link>` в `<head>`.

## Локальный запуск

Просто открой `index.html` в браузере. Или подними локальный сервер:

```bash
python3 -m http.server 8000
# затем открой http://localhost:8000
```

## Деплой

### GitHub Pages
1. Закоммить файлы в репозиторий
2. Settings → Pages → Source: `main` branch, `/` (root)
3. Готово — сайт по адресу `https://<user>.github.io/<repo>/`

### Netlify / Vercel / Cloudflare Pages
1. Подключи репозиторий
2. Build command: оставить пустым
3. Output directory: `/`
4. Деплой

## Языки

Переключатель RU / UZ / EN в навигации. Язык запоминается в localStorage. Также можно задавать через query-параметр: `?lang=uz`, `?lang=en`, `?lang=ru`.

Чтобы поправить перевод — отредактируй `i18n.js`. Все ключи единые между языками: достаточно изменить значение нужного ключа в нужной языковой секции.
